## Getting Started

For help getting started with Flutter, view online
[documentation](https://flutter.io/).

#### 1. [Setup Flutter](https://flutter.io/setup/)

#### 2. Clone the repo

```sh
$ git https://github.com/devefy/Flutter-Single-Product-Page-UI.git
$ cd productdetail/
```

#### 4. Run the app

```sh
$ flutter run
```
